Exercicio 01

A partir do número entrado pelo usuáriodetermine se o número é par ou ímpar.

Exercicio 02

Apartir de um número digitado pelo usuário determine se ele é divisivel por 7 ou por 3.

Exercicio 03

Faça um código que solicite ao usuário o nome a idade. Apresente a mensagem "Bem vindo nome! você tem XX anos". Caso ele seja maior de 18 anos o programa deve solicitar o CPF e exibir o número do CPF. Caso contrário deve exibir somente uma mensagem dizendo que é menor de idade.
