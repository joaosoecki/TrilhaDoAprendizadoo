TAGS DIFERENTES:

TABLE

Utilizado para a criação de tabelas, com table,tr ou th. Exemplo:

Y X Z
3 2 1
C B A

DEL

Representa uma parte do texto que foi excluída.

Este texto foi excluído

INS

Representa um intervalo de texto que foi adicionado a um documento.

Esta parte de texto foi inserida

SUB

O texto subscrito aparece meio caractere abaixo da linha normal e pode ser renderizado para uma fonte menor.

Esse texto contém subscript text abaixo.

SUP

O texto sobrescrito aparece meio caractere acima da linha normal e pode ser renderizado para uma fonte menor.

Esse texto contém superscript text acima.

BLOCKQUOTE

representa que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo. A URL para a fonte da citação pode ser dada usando o atributo cite

Esta é uma citação tirada do youtube.

Q

Representa que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo

According to Mozilla's website, Crome 2.0 lançou suas ferramentas em 2010.

ABBR

Define uma abreviatura ou um acrónimo

O WHO foi criado em 1948.

ADDRESS

Define as informações de contato do autor/proprietário de um documento ou artigo.

Escrito por IFPR.
Visite-nos em:
Exemplo.com
BR

CITE

Representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

A última ceia de Leonardo da Vinci Data 1498.

BDO

É usado para substituir a direcionalidade atual do texto. Isso faz com que a direcionalidade dos personagens seja ignorada em favor da direcionalidade especificada.

Este texto ficará da esquerda para a direita.

Este texto ficará da direita para a esquerda.

CODE

Usado para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

O HTML botâo tag definida ao clicar no botâo.

PRE

Usado para definir texto pré-formatado.

            Esse texto é um pré elemento
            é exibido em uma largura fixa
            fonte, e preserva
            ambos os espaços e
            linhas quebradas

KBD

É usado para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Aperte Ctrl + V para colar algo (Windows).
